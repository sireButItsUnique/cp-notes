## Log (Precise)
<div className="language-cpp">
```c++
int logn(double n, int base) {
    n--;
    int res = 0;
    if (n < 0) {
        return -1;
    }
    while (n) {
        n /= base;
        res++;
    }
    return res;
}
```
</div>

## Square Multiply
<div className="language-cpp">
```c++
ll modpow(ll a, ll b) {
    if (b < 0) return 0;
    ll res = 1;
    while (b) {
        if (b & 1) res = (res * a) % MOD;
        a = a * a % MOD;
        b >>= 1;
    }
    return res;
}
```
